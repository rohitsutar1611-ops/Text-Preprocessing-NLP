
Mini NLP project for text preprocessing using Python and NLTK
Text Preprocessing Pipeline using NLP Techniques in Python

Project Overview
This project demonstrates a basic Natural Language Processing (NLP) pipeline that preprocesses text using tokenization, stopword removal, stemming, and lemmatization.

Objective
To clean and normalize raw text data for improved NLP and machine learning performance.

Technologies Used
- Python
- NLTK (Natural Language Toolkit)

NLP Techniques Implemented
- Tokenization
- Stopword Removal
- Stemming (Porter Stemmer)
- Lemmatization (WordNet Lemmatizer)
